# ESP32 StreanDeck Project
A custom-built StreamDeck using ESP32 for OBS and Simracing as part of my transition from a software developer into embedded systems and hardware engineering

## Goals
- Learn more practical parts of embedded systems and build on learned theory from my studies
- Build a functional, and in the end professional looking hardware device that could be sellable
- Document the full process
- Explore firmware, PCB and low-level devlopment

## V1 Features:
- 4 Buttons
- OLED Display
- USB Serial COM
- OBS Scene Switching
- Mute Controls
- Desktop Client Software

## Future Ideas:
